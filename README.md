# Тестовое задание: Погодное iOS-приложение

## Цель
Разработать iOS-приложение на Swift с использованием UIKit, полностью программно (без Storyboard), отображающее погодную информацию.

---

## Требования к UI

- Отобразить один экран с погодной информацией:
  - Текущая погода
  - Почасовой прогноз:
    - Показывать оставшиеся часы текущего дня
    - Показывать все часы следующего дня
  - Прогноз погоды на 3 дня
- Обработка состояний:
  - Показ загрузки данных
  - Показ ошибки при сбое запроса с кнопкой повторного запроса
- Дизайн:
  - Ограничений нет, можно ориентироваться на приложение "Погода" от Apple
  - Интерфейс должен быть понятным и логичным

---

## Геолокация и запросы

- Запрашивать разрешение на использование геопозиции пользователя:
  - Если пользователь отказывается – использовать Москву
- Получение данных через API:
  - Текущая погода:  
    ```
    http://api.weatherapi.com/v1/current.json?key=YOUR_API_KEY&q=LAT,LON
    ```
  - Прогноз на 3 дня:  
    ```
    http://api.weatherapi.com/v1/forecast.json?key=YOUR_API_KEY&q=LAT,LON&days=3
    ```

---

## Технические требования

- Язык: Swift  
- Фреймворк: UIKit  
- Интерфейс: полностью программный (без Storyboard)
- Графическая составляющая:
  - Оформление на усмотрение разработчика
  - Должно быть понятным, не перегруженным, соответствовать требованиям ТЗ
